Kelebihan dan kekurangan bahasa pemprograman Javasript


Kelebihan:

  1. JavaScript sisi klien sangat cepat, ini berjalan langsung di dalam browser web karena tidak ada persyaratan kompilasi
  2. Memberikan antarmuka yang lebih kaya ke situs web
  3. Sangat serbaguna
  4. Ini adalah bahasa pemrograman web
  5. Permintaan server situs web berkurang berdasarkan sisi klien
  6. Pembaruan rutin melalui spesifikasi ECMA
  7. Beberapa add-on, seperti Greasemonkey, untuk memperluas fungsionalitas
  8. Implementasi sederhana
  9. Banyak sumber daya dan dukungan komunitas raksasa
  10. Digunakan untuk membangun beragam aplikasi
  11. Bekerja sangat baik dengan bahasa pemrograman lain

kekurangan:

  1. Tidak ada salinan atau metode yang setara
  2. Mengizinkan hanya satu warisan
  3. Saat kode dijalankan pada mesin pengguna, banyak orang memilih untuk menonaktifkan JavaScript karena takut dieksploitasi untuk maksud jahat
  4. Mungkin ditafsirkan berbeda oleh browser yang berbeda
